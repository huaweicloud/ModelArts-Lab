###使用ModelArts实现鸢尾花卉分类预测

iris也称鸢尾花卉数据集，是一类多重变量分析的数据集。数据集包含150个数据集，分为3类，每类50个数据，每个数据包含4个属性。可通过花萼长度，花萼宽度，花瓣长度，花瓣宽度4个属性预测鸢尾花卉属于（Setosa，Versicolour，Virginica）三个种类中的哪一类。

文档包含内容：

1. **./codes**：运行代码。
2. **./data**：试验数据。

具体运行指导文档请参考“[华为云ModelArt帮助文档](https://support.huaweicloud.com/bestpractice-modelarts/modelarts_10_0016.html)”
