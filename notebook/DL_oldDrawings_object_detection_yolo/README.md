

  # 古画物体检测任务

  
  在古画标注任务中，采用社区标注的故宫图片数据集，进行物品检测模型训练。
  
  ## 数据集
  社区标注的故宫图片数据集
  
  ## 模型
  YOLOV3模型
  
  ## 实验环境

  - **环境创建**
  Notebook多引擎，Python3.6， GPU
  - **notebook创建**
  创建notebook时请选择：**TensorFlow 1.13**
